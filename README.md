## Stack

- React (from Create React App)

- Chakra UI
- Apollo Client
- Moment

## Getting Started

1. Clone the repo.
2. `cd` into client and server directories, and `npm i` dependencies
3. Start the server with `npm start-server` and the client with `npm run start`
